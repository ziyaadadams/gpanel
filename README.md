# GPanel - Ultimate MacOS Transformation

GPanel is a powerful GNOME Shell extension that transforms your desktop into a fully-featured macOS-style environment. It is designed for GNOME 46 and 47.

## Features

### üñ•Ô∏è MacOS-Style Panel
- **Global Menu Integration**: Shows the active application name and menu items (File, Edit, View...).
- **Traffic Lights**: Simulated window controls (Close, Minimize, Maximize) directly on the panel.
- **Blur & Transparency**: Configurable background blur and opacity.
- **Custom Height**: Adjust the panel height to your liking.

### üèùÔ∏è Dynamic Notch
- **Dynamic Island**: The clock area transforms into a notch that expands on hover.
- **Animation**: Smooth elastic animations for a premium feel.
- **Configurable**: Toggle the notch or adjust its width.

### ‚öì Built-in Dock
- **MacOS Dock**: A fully functional dock at the bottom of the screen.
- **Favorites**: Automatically populates with your pinned apps.
- **Animations**: Smooth hover effects and scaling.
- **Autohide**: Optional autohide behavior.

### ‚öôÔ∏è Customization
- **Preferences Window**: Built with Libadwaita for a native look.
- **Toggles**: Enable/disable features like Traffic Lights, App Menu, Notch, and Dock.
