## ğŸ’¡ Visual Tips

### For macOS Look
- Enable ALL features
- Use rounded corners (12px)
- Set blur to ON
- Panel height 26-28px
- Dock icons 48-56px

### For Clean Look
- Disable notch
- Disable traffic lights
- Straight corners
- Lower opacity

- Disable blur
- Disable magnification
- Smaller icons
- No auto-hide
